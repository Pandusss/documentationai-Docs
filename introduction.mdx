## Quick Start

Get up and running with Doulot Doulottt in minutes. Follow these steps to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://doulotdoulottt.com` and create a free account using your email.
  </Step>
  <Step title="Create Workspace" icon="plus">
    Click "New Workspace" and name it for your project, like "My Project Docs".
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Select "New Page" and start writing in Markdown or our visual editor.
  </Step>
  <Step title="Invite Team" icon="mail">
    Share your workspace link or add members via email for collaboration.
  </Step>
</Steps>

<Callout kind="tip">
  Customize your theme early by setting the primary color to `{`#3B82F6`}` in workspace settings. This ensures all docs reflect your brand consistently.
</Callout>

## Example Integration

Embed Doulot Doulottt pages in your app using our API.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/docs/embed', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
    body: JSON.stringify({ pageId: 'doc-123' })
  });
  const embedUrl = await response.json();
  ```
  ```python
  import requests
  response = requests.post(
      'https://api.example.com/docs/embed',
      headers={'Authorization': 'Bearer YOUR_API_KEY'},
      json={'pageId': 'doc-123'}
  )
  embed_url = response.json()['url']
  ```
</CodeGroup>
